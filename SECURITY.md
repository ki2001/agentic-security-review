# Security Policy

Agentic Security Review is a defensive, authorized code-review workflow.

## Authorized use only

Use this project only on:

- code you own;
- code you maintain;
- open-source repositories where static review is allowed;
- private repositories where you have explicit authorization.

Do not use it to attack third-party systems or deployed services.

## Default boundaries

The default workflow is static/local review only. It should not perform live exploitation, network scanning, credential theft, persistence, malware generation, or destructive testing.

## Reporting vulnerabilities in this project

If you find a vulnerability in this repository, please open a private security advisory or contact the maintainer. Do not publish exploit details before maintainers have had a reasonable chance to respond.
